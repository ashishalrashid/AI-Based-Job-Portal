HR App Frontend Documentation
==============================

Welcome to the **HR Recruitment System Frontend** documentation. This guide covers the Vue.js 3 single-page application, component architecture, state management, routing, and user interface patterns.

The frontend is built with:
- **Vue 3** - Progressive JavaScript framework
- **Vite** - Next-generation frontend build tool
- **Vuex** - State management library
- **Vue Router** - Client-side routing
- **Axios** - HTTP client for API communication
