HR App Backend Documentation
============================

Welcome to the **HR Recruitment System Backend** documentation. This comprehensive guide covers the Flask-based REST API, database models, authentication system, and all backend services that power the HR recruitment platform.

The backend is built with:
- **Flask** - Web framework for RESTful APIs
- **SQLAlchemy** - ORM for database operations
- **Flask-Security** - Authentication and authorization
- **JWT** - Token-based authentication
- **Redis** - Caching and Celery task queue
- **Gunicorn** - Production WSGI server
